Projects

Arcade Game

The visuals where created in an arcadeish style to provide the user with nostalgia of the old arcades. The style was inspired from the pixel games from that time. The change of color schema used for this project provides the player with visual feedback throughout the game. The bright colors are aimed to draw people to the game while other players are interacting with it.

Point And Click Game

The semi realistic-cartoony style aims at our younger audience. The items are purposely hidden around the non-collectable ones and drawn almost identically to make the user think what he has to collect.

Physics Game

The stylization of the visuals for this project are influenced by the age group we are aiming at (4-12 years). The old cartoon type of vibe is provided to give nostalgia to the parents and a fun experience for the kids. Sepia effect is used throughout the game with Glitchy animations to enhance the feeling of an old factory. The simplicity of the art style is our main goal so we can make everything understandable and clear. The facial expressions are made cute and expressive, so the user can feel empathy for the characters.
